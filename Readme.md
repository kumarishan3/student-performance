Student Performance Prediction (Machine Learning Project)

Problem

Predict whether a student will pass or fail based on study hours and attendance.

 Dataset
The dataset contains:
- Study Hours
- Attendance
- Pass (0 = Fail, 1 = Pass)

 Approach
- Loaded data from CSV file
- Performed data preprocessing and scaling
- Used train-test split with stratification
- Trained two models:
  - Logistic Regression
  - Decision Tree
- Compared performance using accuracy and confusion matrix

 Results
- Achieved high accuracy on test data
- Clear separation between pass and fail cases

 Key Learnings
- Understanding classification problems
- Model comparison
- Evaluation using confusion matrix

 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
